# 🔐 Multi-function Web Security Tool

## 📌 Overview

This project is a web-based security tool designed to help users analyze, validate, and improve their cybersecurity practices. It combines multiple security features into a single platform.

---

## 🚀 Features

### ✅ Input Validator

* Validates emails, URLs, and user inputs
* Detects potentially unsafe or malformed data

### 🔍 Breach Checker

* Checks if passwords have been exposed in known data breaches
* Uses **k-anonymity** to ensure passwords are never sent in plaintext

### 🌐 Port Scanner

* Scans for open ports on a target system
* Helps identify potential vulnerabilities

### 📡 Traffic Analyzer

* Captures and analyzes network traffic
* Supports filtering by:

  * Source IP
  * Destination IP
  * Protocol (TCP, UDP, ICMP)

### Frontend

* React (Vite)

### Backend
